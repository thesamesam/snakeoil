snakeoil.descriptors module¶

Classes implementing the descriptor protocol.

class snakeoil.descriptors.classproperty(getter)[source]¶

Bases: object

Like the builtin property() but takes a single classmethod.

Essentially, it allows you to use a property on a class itself- not just on its instances.

Used like this:

>>> from snakeoil.descriptors import classproperty
>>> class foo:
...
...   @classproperty
...   def test(cls):
...     print("invoked")
...     return True
>>> foo.test
invoked
True
>>> foo().test
invoked
True
